# DOGTI 狗格测试 — 执行计划

## 概述
做一个类似 SBTI 的 H5 单页问答，20 道题测出用户的「狗格类型」——MBTI 16 型人格对应的 16 种狗。

## 技术选型
- 单文件 HTML + CSS + JS，无框架依赖
- 仿 SBTI 的移动端卡片式 UI
- 所有题目一次性展示（上下滚动），全做完才能提交

## 文件结构
```
dogti.html    — 唯一文件，包含全部逻辑
```

## 核心逻辑

### 计分规则
- 每题 3 个选项：倾向 A（+1）、中性（0）、倾向 B（-1）
- 四个维度各 5 题，单维度总分范围 -5 ~ +5
- 总分 > 0 → 第一字母，总分 ≤ 0 → 第二字母（0 时默认第二字母）
- 组合 4 个字母得到 MBTI 类型，映射到对应犬种

### 维度 → 字母映射
| 维度 | 正分 | 负分/零分 |
|------|------|-----------|
| E/I | E | I |
| S/N | S | N |
| T/F | T | F |
| J/P | J | P |

### 16 型 → 犬种映射
```
INTJ → 边牧      INTP → 柴犬
ENTJ → 德牧      ENTP → 哈士奇
INFJ → 阿富汗猎犬  INFP → 灵缇
ENFJ → 金毛      ENFP → 西高地
ISTJ → 秋田犬    ISFJ → 纽芬兰
ESTJ → 美国斗牛犬  ESFJ → 伯恩山
ISTP → 比利时马犬  ISFP → 巴哥
ESTP → 腊肠犬    ESFP → 博美
```

## UI 设计

### 配色
- 主色调：暖黄色/奶油色（狗毛质感）
- 辅助色：深棕色文字
- 按钮：圆角、大 touch target

### 题目页
- 顶部：进度条 + 题号（仿 SBTI："0 / 20"）
- 标题：DOGTI 狗格测试
- 每个维度一个区块，区块标题显示维度名（可隐藏具体含义，只显示 "Part 1" 等）
- 每题显示题干 + 3 个 radio 选项卡片
- 底部固定按钮：「提交并查看结果」（全部选完才可点击，否则置灰 + 提示文案）

### 结果页
- 大图占位区（用户后续提供图片路径，先用 CSS 占位 + 标注）
- 狗种名称 + 英文名
- MBTI 类型标签（如 "你的狗格：ENFP"）
- 一句狗种描述文案（用占位文字）
- 「重新测试」按钮

## 实施步骤

### Step 1: HTML 结构
- 题目列表（20 题，分 4 个维度区块）
- 结果页容器（默认隐藏）
- 每个 radio 附带 data 属性标记所属维度和分值

### Step 2: CSS 样式
- 移动端优先，max-width 480px 居中
- 卡片式题目，选中态高亮
- 进度条动画
- 按钮状态切换

### Step 3: JS 评分逻辑
- 监听 radio change，检查是否全部答完
- 全部答完启用提交按钮
- 提交后遍历所有选中项，累加四个维度分数
- 判定 4 个字母，查表得犬种
- 切换显示结果页

### Step 4: 结果页渲染
- 根据犬种展示对应内容
- 图片 src 预留为可配置变量（后续替换路径）
- 重新测试按钮重置全部状态

## 图片占位
- 结果页图片位置留空，使用 CSS 占位符
- 图片路径集中在一个配置对象中，方便后续批量替换
- 尺寸：约 240x240px 的圆形或圆角方形区域
